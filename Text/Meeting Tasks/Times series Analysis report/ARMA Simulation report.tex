\section{Purpose}
The main purpose of this work is to discriminate between autoregressive, moving average, and ARMA time series models using feature based ordinal pattern. These information metrics help identify models more effectively than traditional linear statistics because they can detect both linear and nonlinear structure. \cite{Boaretto2021, Chagas2022a, Rey2024, Rey2023} 


\section{Methodology}
The results involving the time delay parameter variation is not significantly effect in repeated experiment~\cite{Chagas2022a} we consider the two relevant variables: length of the sequence and the embedding dimension to set parameters and data set. Here are the simulation setup for our experiment. 	


\textbf{Simulation and Feature Construction:}
	\begin{itemize}
		\item \textbf{Embedding:} Fixed embedding dimension ($D=3$), delay ($\tau=1$).
		\item \textbf{Model structure:} AR(1), AR(2), MA(1), MA(2), ARMA(1,1), ARMA(2,2), each with four parameterizations (M1-M4).
		\item \textbf{Simulation:} For each model and each parameter set, $R=100$ time series of length $n=1000$ are simulated.
		\item \textbf{Features:} For each realization:
		\begin{itemize}
			\item \emph{Entropy:} Shannon, Rényi, Tsallis, and Fisher information entropies.
			\item \emph{Complexity:} Statistical complexity measures matched to each entropy type.
			\item \emph{Standard deviations:}  Variances for each model were considered.
		\end{itemize}
	\end{itemize}
	
	\noindent
	\textbf{Centrality analysis:} For each model, the series with $(H^*, C^*)$ closest to the median $(H, C)$ is identified as the ``central'' representative as recommended by Chagas et al.~\cite{Chagas2022a}.
	
\section{Results}
	
\subsection{Median (H*, C*) points}
	
To calculate the "median or emblematic" point in the $H \times C$ plane (as recommended by Chagas et al.,~\cite{Chagas2022a}), first compute the principal component analysis (PCA) of all H and C points for a given model. Next, sort the data according to the values of the first principal component (the direction of greatest variance). The emblematic point is then the original $(H,C)$ pair corresponding to the median value along this first principal component. This method ensures the selected point is truly central with respect to the main data structure, rather than simply taking the median of H and C separately.

\subsection{Central Series Exploration}
Refered to the chagas et.al~\cite{Chagas2022a} it is clear with that, median point for entropy and complexity is the best representation of the time series. In our experiment, the emblematic  $(H^*,C^*)$ point for each model is identified following Chagas et al.~\cite{Chagas2022a}, as the $(H,C)$ point whose first principal component is the median among all observations. This ensures the selected point is most representative in the main direction of data variability, as recommended in Algorithm 1 and Figure 6 of Chagas et al.
 	
The figure~\ref{fig:median_entropy} displays all median $(H^*, C^*)$ points for every model on a single entropy-complexity plane, with each point colored by entropy type (Shannon, Renyi, Tsallis, or Fisher). This collective view reveals the global clustering structure and relative discriminability achieved by the different entropic measures. Further it is clearly see that different models occupy distinct regions in the $H \times C$ plane. Models with highly regular or highly random dynamics tend to lie near the lower and upper bounds of $H^*$ and $C^*$, while those with intermediate complexity are centrally located. Coloring by entropy type allows for immediate visual comparison of how sensitive each measure is for model separation.

The second figure~\ref{fig:median_facet} facets the median points into separate panels for each entropy type. This layout enables a detailed, side-by-side assessment of how the choice of entropy influences the position and clustering of models in the entropy-complexity plane. Some entropy types—such as Tsallis and Fisher reveal stronger model separation or different spatial ordering compared to Shannon and Renyi, reflecting their distinct sensitivities to aspects of time series structure. This faceted view highlights which entropic measure best distinguishes among the models under consideration and supports nuanced conclusions about the comparative performance of each metric in model discrimination tasks.

\subsection{Exploratory Data Analysis and Statistical Testing}
	
Main feature discriminability was examined using pair plots, ANOVA, Kruskal--Wallis, PCA, and Random Forests (feature ranking):
	
	\begin{itemize}
		\item ANOVA and Kruskal-Wallis confirm model-wise statistical separability of entropies and complexities (most $p < 0.01$).
		\item PCA of all features reveals group separability primarily along the first principal component(s).
		\item Fisher, Rényi, and Tsallis complexities provide additional discrimination in non-Gaussian regimes.
		\item Random forest feature importance corroborate entropy variance and hybrid complexity as top ranking.
	\end{itemize}
	
\section{Discussion}
	
The entropy-complexity approach, especially when leveraging diverse metrics (Shannon, Rényi, Tsallis, Fisher), enables clear discrimination of model structure for clustered (H,~C) patterns and central series. Advanced visualization (contours, ellipses, custom color maps) improves both interpretability and communication of results. Limitations (model overlap for certain parameter regimes) and alternative visualizations (including segment or quantile-based time series plots) are acknowledged.
