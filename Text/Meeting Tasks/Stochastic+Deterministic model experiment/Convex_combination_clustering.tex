\section{Purpose}
\noindent
The main purpose of this research is to use the simulated results from the previous study to support time series clustering. The simulation results show that the $H \times C$ representation captures two broad regimes of behavior: lowâ€‘entropy, highâ€‘complexity dynamics and highâ€‘entropy, lowâ€‘complexity dynamics. The diversity exhibited across these regimes confirms that ordinal pattern features provide meaningful discrimination between qualitatively different time series models. This separation in the $H \times C$ plane, together with the additional features derived from R\'enyi entropy, Tsallis entropy, and Fisher information, provides an enriched feature space that yields a more detailed description of each time series.

\medskip

\noindent
Based on these features, we apply unsupervised time series clustering. The goal is to determine whether time series generated by different mechanisms can be grouped correctly using only their ordinal pattern features and $H \times C$ plane representations. The clustering results are evaluated using visual inspection of the $H \times C$ plane and silhouette coefficients from the $k$-means algorithm.

	
\section{Methodology}{\label{sec:methodolpgy}}


In this experiment, we consider two deterministic signals (sine,logistic map), and a stochastic model  (ARMA, including two particular cases, namely AR and MA). Based on the outcomes of the simulation study, we fixed the model parameters. The following section describes the simulation setup and feature construction procedures used in our clustering experiment.

\textbf{Simulation and Feature Construction:}
\begin{itemize}
	\item \textbf{Embedding:} A fixed embedding dimension ($D = 4$) and time delay ($\tau = 1$) are used.
	
	\item \textbf{Model structure:} The models considered include $\mathrm{AR}(2)$, $\mathrm{MA}(2)$, $\mathrm{ARMA}(2,2)$, the sine function $f(t)$, and the logistic map $L_t$.
	
	\item \textbf{Simulation:} For each model and parameter set, $R = 50$ time series are simulated with lengths $n = 1000$ and $n = 10000$.
	
	\item \textbf{Features:} For each realization, the following quantities are computed:
	\begin{itemize}
		\item \emph{Entropy:} Shannon, RĂ©nyi, and Tsallis entropies, along with the Fisher information measure.
		\item \emph{Complexity:} Statistical complexity measures corresponding to each entropy.
		\item \emph{Variance:} Variances for each model to assess the stability of the extracted features.
		\item \emph{Confidence intervals:} Confidence intervals for entropy and complexity to further evaluate feature stability.
	\end{itemize}
	The experimental results is based on the following fourteen ordinalâ€‘pattern features:
	\begin{itemize}
		\item $H_{\text{Shannon}}$, $H_{\text{RĂ©nyi}}$, $H_{\text{Tsallis}}$, $H_{\text{Fisher}}$,
		\item $C_{\text{Shannon}}$, $C_{\text{RĂ©nyi}}$, $C_{\text{Tsallis}}$, $C_{\text{Fisher}}$,
		\item Disequilibrium,
		\item $\mathrm{Var}(H_{\text{Shannon}})$, $\mathrm{Var}(H_{\text{RĂ©nyi}})$, $\mathrm{Var}(H_{\text{Tsallis}})$, $\mathrm{Var}(H_{\text{Fisher}})$,
		\item $\mathrm{Var}(C_{\text{Shannon}})$.
	\end{itemize}
	
	\item \textbf{Clustering:} Silhouette coefficients are used to assess the quality of clustering obtained via the $k$-means algorithm.
\end{itemize}

\noindent
For any stochastic model $\{X_t^{(S)}\}$ and deterministic model $\{X_t^{(D)}\}$, 
we define a convex combination as
\[
X_t(w) \;=\; w\,X_t^{(S)} + (1-w)\,X_t^{(D)}, 
\qquad 0 \le w \le 1,
\]
where the weight parameter $w$ controls the relative contribution of the stochastic and deterministic components.

\noindent
In our experiments, $X_t^{(S)} \in \{\mathrm{ARMA}(2,2), \mathrm{AR}(2), \mathrm{MA}(2)\}$ 
and $X_t^{(D)} \in \{\text{Logistic map } (L_t), \text{sine } f(t)\}$.
For each pair $(X_t^{(S)}, X_t^{(D)})$, the mapping
\[
w \longmapsto \bigl(H(X(w)),\, C(X(w))\bigr)
\]
traces the evolution of entropy and complexity coordinates as $w$ varies.

\noindent

We examined the correlations among the fourteen ordinalâ€‘pattern features used in this study. The correlation matrix (Figure~\ref{fig:correlation_D4}) $(n = 10000)$ reveals several strong dependencies among the 14 ordinal-pattern features. A detailed list and description of all fourteen features is given in the Section~\ref{sec:methodolpgy}. 
The three entropy measures ($H_{\text{Shannon}}$, $H_{\text{RĂ©nyi}}$, $H_{\text{Tsallis}}$) exhibit extremely high positive correlations ($r \approx 0.99$), indicating that they capture nearly identical distributional information. 
A similar pattern appears among the statistical complexity measures ($C_{\text{Shannon}}$, $C_{\text{RĂ©nyi}}$, $C_{\text{Tsallis}}$), which show correlations in the range $0.98$-$0.99$. 
Strong negative correlations are observed between the entropy measures and disequilibrium 
($r=-0.99$ for $H_{\text{Shannon}}$ vs.\ Disequilibrium). Entropy increases when the pattern distribution becomes more random, whereas disequilibrium increases when the distribution becomes more uneven. 
Because these behaviours oppose each other, the two measures are strongly negatively correlated.
Variance based features also display high internal correlation, particularly between $\mathrm{Var}(H_{\text{RĂ©nyi}})$ and $\mathrm{Var}(H_{\text{Tsallis}})$ ($r=0.96$). 
In contrast, several feature pairs exhibit weak associations: Fisher entropy shows only small correlations with entropy variances ($|r| < 0.20$), and $\mathrm{Var}(C_{\text{Shannon}})$ presents generally low correlations with other measures. 
Overall, the matrix indicates substantial multicollinearity among entropy and complexity based features, while Fisher related and variance based descriptors provide more independent information for clustering.

\section{Time Series Clustering}
Clustering is an unsupervised learning process that involves organizing a collection of patterns into similar groups based on their similarity. 
The term \textit{unsupervised} refers to constructing a model or partitioning observations into groups directly from the data, without prior labels. 
The term \textit{learning} indicates that the statistician calibrates models to the available data. 
Once clusters or groups are defined, supervised learning can be applied, where \textit{supervised} means that the group membership is known a priori. 
Clustering is one of the most important problems in data mining and has numerous applications across various fields such as the natural sciences, the medical sciences, economics, marketing. In the context of time series analysis, clustering groups time series with similar temporal patterns, dynamics, or statistical properties.
Once clusters are obtained, it is generally useful to describe and interpret each group. 
Identifying compact and well-separated clusters in data, however, remains a challenging task.


As discussed in the book note in "Applied Multivariate Statistical Analysis"~\cite{HardleSimarFengler2024}, clustering reveals latent structure in high-dimensional data, making it fundamental for exploratory analysis.
In this research work, we focus on simulated time series clustering using the $k$-means algorithm and validate the results using silhouette coefficients.

\subsection{Simulation study to assess the clustering procedure}

To evaluate the performance of the clustering approach, time series were simulated from a collection of fifteen time series, including the convex combination models based on $\mathrm{ARMA}(2,2)$, $\mathrm{AR}(2)$, $\mathrm{MA}(2)$, Logistic, Sine, and their hybrid extensions as described in the Table~\ref{tab:series_list}. 
For each model class, multiple series of lengths 1000 and 10000 were generated, and the clustering procedure was applied. 
This entire process was repeated 50 times in order to evaluate the stability of the clustering outcomes.  

For each repetition, we recorded the number of correctly identified clusters, where a correct cluster is defined as one containing all series generated from the same underlying process. 

The clustering is based on the following fourteen ordinalâ€‘pattern features:
\begin{itemize}
	\item $H_{\text{Shannon}}$, $H_{\text{RĂ©nyi}}$, $H_{\text{Tsallis}}$, $H_{\text{Fisher}}$,
	\item $C_{\text{Shannon}}$, $C_{\text{RĂ©nyi}}$, $C_{\text{Tsallis}}$, $C_{\text{Fisher}}$,
	\item Disequilibrium,
	\item $\mathrm{Var}(H_{\text{Shannon}})$, $\mathrm{Var}(H_{\text{RĂ©nyi}})$, $\mathrm{Var}(H_{\text{Tsallis}})$, $\mathrm{Var}(H_{\text{Fisher}})$,
	\item $\mathrm{Var}(C_{\text{Shannon}})$.
\end{itemize}

Both visual inspection of the clustering patterns in the $H \times C$ plane and the silhouette based evaluation are used to compare and validate the algorithmic results.
